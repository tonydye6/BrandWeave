# BrandWeave

> **University brand compliance — skin generation, pre-screening, and approval workflows**

Part of the [SparqPlaymaker™](https://github.com/tonydye6/SparqPlaymaker) platform — Wave 2 — Content & Compliance

---

## Overview

BrandWeave manages the relationship between game content and university brand standards. It handles brand data management, UI skin generation, automated pre-screening (8 checks), human review workflows, and Brand Manager 360 integration.
